URL API LAPAR FOR USER :

Proses Login
URL 		: http://localhost/lapar/user/v1/login
Method 		: POST
Parameter	: Email, Password

Proses Registrasi
URL		: http://localhost/lapar/user/v1/register
Method		: POST
Parameter	: Name, Email, Password, Mobile

Proses Update Profile
URL		: http://localhost/lapar/user/v1/profile_update
Method		: POST
Parameter	: Name, Email

Proses Get RM   
URL             : http://localhost/lapat/user/culinar
Method		: GET

Proses Get RM & Menu
URL             : http://localhost/lapat/user/culinar/:id
Method		: GET
